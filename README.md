# 숫자 야구 게임
* 요구사항   
  * 프로그램이 1 ~ 9 까지의 임의의 숫 3개를 정하고,         
    플레이어가 맞추는 게임이다.        
  
  * 정답을 맞추기 위해 3자리수를 입력하고 힌트를 받는다.      
  
  * 힌트는 야구 용어인 strike, ball 로 표현한다.         
  
  * 같은 자리, 같은 숫자라면 `strike`,          
    다른 자리, 같은 숫자라면 `ball`          
  
  * 입력한 숫자와 임의의 숫자가 같다면 게임 종료
